Presentation.js

About this library

Presentation.js is a javascript library which can create the slide using html, css, and javascript for presentations.
By making "not to take a user's freedom" into a concept, it is designing on the assumption that it can extend.

The feature of a library

Some are following as a big feature.

  • Offer of the plugin extension using a helper.
  • Offer of the optimization facility of contents.
  • Offer of the initialization function of a slide.
  • iPad correspondence using the library of third party nature.

The function of Presentation.js

Three big features

This library is made on the assumption that it extends, as the outline of Presentation.js explained.

I introduce three big features one in order also in it.

The feature 1 - Plugin using a helper

Extension by inclusion of plugin

Helper plug-in is expanded function freely incorporable according to the contents of the presentation.
The plugin currently prepared from the beginning is as follows.

  • Controller - The GUI interface which can operate a slide is offered.
  • Page - The page information displayed now is displayed.
  • Keyboard - Operation of the slide by a keyboard input is enabled.
  • Swipe - A slide is made to correspond to tap operation with a mobile terminal.

The plugin itself can be created very easily.

The demo of a helper is seen.

The feature 2 - Powerful contents filter

Optimization of the contents using a filter

If a filter is used, contents can be optimized freely.
A filter is applied, when displaying applicable contents, and when making it undisplayed.

To this timing, some contents can be substituted or new contents can be added.

The demo of a filter is seen.

The feature 3 - Initialization of the slide by bootstrap

A slide is optimized freely.

If a bootstrap function is used, a slide can be initialized freely.
For example, the contents of the slide can be dynamically read by Ajax, or its own plug-in can be incorporated.

Moreover, since initialization processing can be modularized, it is possible to create a slide setup for every case to be used.

The demo of a bootstrap is seen.
